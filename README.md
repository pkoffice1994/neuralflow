# NeuralFlow

Is project ko `Vercel` par deploy karne ke liye database ko `GitHub` me save mat karo. `GitHub` sirf code ke liye use hota hai. Persistent data ke liye `Supabase` use karo.

## 1. Supabase table banao

Supabase SQL editor me yeh query chalao:

```sql
create table if not exists leads (
  id bigint generated by default as identity primary key,
  name text not null,
  email text not null,
  created_at timestamp with time zone default now()
);
```

## 2. Env variables set karo

`.env.local` file banao:

```bash
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

## 3. Local run

```bash
npm install
npm run dev
```

## 4. GitHub + Vercel

1. Is folder ko GitHub repo me push karo
2. Repo ko Vercel se import karo
3. Vercel project settings me same env variables add karo
4. Deploy karo

## Important

- `GitHub` me database file ya secret keys commit mat karo
- `Vercel` ka local filesystem permanent database ke liye reliable nahi hota
- `Supabase`, `Neon`, ya `Vercel Postgres` jaise hosted database best rahenge
